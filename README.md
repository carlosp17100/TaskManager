# TaskManager

Full-stack Task Manager built with vanilla JavaScript, Express and SQLite.

## Features

- Create, update, delete tasks
- Mark tasks as completed
- Filters and search
- Pagination
- Task statistics (total, done, pending)

## Tech Stack

- Node.js
- Express
- SQLite
- Vanilla JavaScript
- HTML + CSS

## Project Structure

Task-manager/
public/
index.html
styles.css
app.js
server.js
package.json

## Run Locally

```bash
npm install
node server.js
```

http://localhost:3001
